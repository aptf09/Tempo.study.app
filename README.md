# Tempo.study.app

Description / Vision

The AI Study App is designed to create tailored study plans for students. It uses AI to optimize learning based on:

Student’s syllabus and subjects taken
Current grades or marks
Desired outcomes and timelines

The app tracks:

Study times
Progress towards goals
Provides motivation and productivity tips to prevent burnout

It also features a built-in AI helper to answer questions, give guidance, and suggest adjustments to study plans.

Current Version (25/02/2026) : Only the welcome screen, quote screen, name input, and login selection are implemented. All AI-powered features are planned and under development.
